# Automated Whitelist Merger

Инструмент для автоматического сбора и объединения белых списков (whitelists) из различных баз данных в готовые подписки.

## 📌 Описание проекта
Скрипт автоматически отслеживает изменения, скачивает актуальные белые списки из множества внешних источников и консолидирует их. Это полностью исключает ручную работу и гарантирует актуальность данных в реальном времени.

## 🚀 Основные функции
* **Полная автоматизация**: сбор данных по расписанию без участия пользователя.
* **Валидация**: строгая проверка строк на корректность форматов.

## 📂 Доступные подписки
В проекте генерируются два варианта списков:
* **`reality`** — специализированный белый список для протокола REALITY. (https://raw.githubusercontent.com/andreykorolev541-dotcom/vless/main/configs/working_reality_sub.txt)
* **`all`** — общий объединенный список, включающий все собранные базы данных. (https://raw.githubusercontent.com/andreykorolev541-dotcom/vless/main/configs/working_sub.txt)

## ⚙️ Инструкция по использованию в VPN-клиенте

Вам достаточно один раз добавить ссылку на нужную подписку в ваше приложение, и список будет обновляться сам.

1. **Скопируйте прямую ссылку** на нужный итоговый файл (`reality` или `all`). Используйте `raw` ссылку из вашего репозитория.
2. **Откройте настройки** вашего VPN-клиента (например, Nekobox, Amnezia, Shadowrocket, Sing-box).
3. **Вставьте скопированную ссылку** в поле для URL-адреса списка.
4. **Включите автообновление** в настройках клиента, чтобы всегда иметь актуальные данные.
